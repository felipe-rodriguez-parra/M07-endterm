# SpaceApps

> Your complete guide to the SpaceX universe

Discover the incredible fleet of rockets that is revolutionizing space exploration. SpaceApps offers you instant access to detailed information about all SpaceX launch vehicles.

---

## Main Features

### Complete rocket catalog
Explore all SpaceX rockets. Each rocket includes detailed technical specifications.

### Real-time information
Data updated directly from the official SpaceX API. Stay up to date with the latest information.

### Detailed technical sheets
- Vehicle height and mass
- Cost per launch
- First flight date
- Operational status (active/retired)
- Official image gallery

### Offline mode
Data is stored locally so you can access information even without an internet connection.

### Immersive space design
Modern interface with space-inspired dark theme. Enjoy a visual experience that transports you to the cosmos.
